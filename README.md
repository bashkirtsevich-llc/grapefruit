## Installation
1. Download grapefruit installer repository
```bash
git clone https://github.com/bashkirtsevich/grapefruit.git
cd grapefruit
```
2. Install Docker (optional)
```bash
# For Centos-7
sudo ./docker-install.sh
```
3. Build & run all
```bash
./install.sh
```

## Components
* mongodb
* nginx
* grapefruit-crawler
* grapefruit-web
